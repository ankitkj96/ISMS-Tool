Controls

List of controls

This screen is used to display the list of controls and to filter them by:

  • domain,

  • attribute,

  • planning period,

  • control status: all, done and to do,

  • free text on name, purpose, comments.

When you click:

  • The domain, you arrive at the security domain display screen

  • Clause, you arrive at the security measure display screen.

  • The date of completion, planning or the date of the next check, you arrive at the security check display screen.

Show a control

This screen contains the information of a control:

  • The name of a control;

  • The objective of the control;

  • Attributes ;

  • Data;

  • The model ;

  • The dates of planning, completion and the date of the next inspection;

  • The score assigned to the control; And

  • The score attributed to the control (green, orange or red).

The "Do" and "Schedule" buttons are present if this control has not yet been carried out.

The "Modify" and "Delete" buttons are available if the user is an administrator.

When you click:

  • "Make", you are sent to the control completion screen

  • "Plan", you are sent to the control planning screen

  • "Edit", you are sent to the control edit screen

  • "Delete", the control is deleted and you are sent to the list of controls.

  • "Cancel", you are sent to the list of controls

Schedule a control

This screen is used to schedule a control.

This screen contains the information of a control:

  • The name of a control;

  • The objective of the control;

  • The date of palnification;

  • Periodicity ; and

  • Those responsible for carrying out the controls.

When you click:

  • "Plan", the planning date, the recurrence and the persons in charge are updated and you are returned to the display screen of controls

  • "Cancel", You are returned to the control display screen

Making a check

This screen allows you to perform a security check.

This screen contains:

  • The name of the control

  • The goal

  • Data

  • Completion date, planning date

  • Control observations

  • A field to save evidence

  • A link to download the control sheet

  • The computation model applied

  • The note

  • The score

  • The action plan

  • The date of the next check

When you click:

  • "Do", the control is saved and a new control is created on the scheduled date

  • "Save", the control is saved

and you return to the list of controls.

Control sheet

The check sheet is a word document generated by the application based on the check data.

This form allows you to describe the observations made, to add screenshots or tables and to electronically sign the observations.

The checklist can be adapted to match your organization's document model.
