Security measures

This part allows you to define, modify and plan new security measures.

List of security measures

The list of security measures allows you to display the list of measures, to filter them by domain or to search for a measure based on part of its name.

Clicking on :

  • the domain, you arrive at the chosen domain definition

  • the clause, you arrive on the description of the security measure

  • the date you arrive the security measure planning screen

Show security measure

This screen displays a security measure.

A security measure consists of:

  • of a domain;

  • of a name;

  • a security objective;

  • attributes;

  • measurement data:

  • of the verification model;

  • an indicator (green, orange, red); and

  • an action plan to be applied if control of this security measure fails.

When you click:

  • “Plan”: you arrive at the security measure planning screen.

  • "Modify": you arrive at the measurement modification screen

  • “Delete”: allows you to delete the measure and return to the list of security measures

  • "Cancel": returns you to the list of security measures

Edit a security measure

This screen allows you to modify a security measure.

When you click:

  • "Save", the changes are saved and you return to the security measure display screen.

  • "Cancel", you return to the security measure display screen.

Schedule a control

This screen is used to plan a control based on a security measure.

The screen contains:

  • The domain ;

  • The name of the measure;

  • The security objective;

  • The planning date; and

  • The frequency of checks;

  • Those responsible for carrying out the controls.

When you click:

  • "Plan",

    • If there is no control for this measure, a new control is created by copying all the data from the measure and is scheduled on the specified date.

    • If there is already an uncompleted check for this security measure, the planning date is updated.

You then return to the list of security measures.

  • "Unplan", the security check associated with this measure is deleted and the user returns to the list of security measures.

  • "Cancel", you return to the list of security measures.
